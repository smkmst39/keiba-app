## 6. 三連複・三連単のサンプル不足について

閾値が強いため、1レースで EV≥1.05 の馬が 3頭以上出るケースは 795R 中わずか **2レース**。三連系はデータ不足で評価不能。

対策案（将来）:
1. **三連系は閾値を緩めた別戦略**: EV上位3頭 BOX のまま残す（現行 56.7% / 53.9%）
2. **閾値を段階化**: 単勝・馬連は 1.05、三連系は 1.00 などと券種で変える
3. **フォーメーション**: 閾値通過馬1頭 + 他は広く流す
