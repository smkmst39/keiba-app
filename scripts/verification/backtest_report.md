### 最適な CORRECTION_FACTOR

- **構造的知見**: 全係数で総合回収率が完全一致 (56.7%)。これは「EV 上位 N 頭の BOX 買い」戦略が順位のみを使うため、CORRECTION_FACTOR がレース内の全馬に等しい倍率を掛ける限り並び順が変わらないことに起因する。係数最適化を意味あるものにするには、① 「EV≥1.0 の馬のみ購入」等の閾値ベース戦略、または ② スコアの低い馬のオッズ再評価 (calcAdjProb の非線形化) が必要。
- 現行値 0.2 は `lib/score/CLAUDE.md` の制約 (0.3超禁止) 内で中央値。本バックテストからは現行維持で問題なし。

- スコア1位が1着に来る率が 11.8% と低い (JRA平均頭数14頭では単勝ランダム = 7.1%)。スコア式自体の予測力がまだ限界で、特に 上がり3F (25%) と 同コース成績 (20%) の組み合わせは「過去の速さ」指標に偏りすぎている可能性。
